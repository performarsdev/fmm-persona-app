# Product Overview

This is a HubSpot Private App template using static authentication. The app extends HubSpot CRM functionality with custom components.

## Core Features

- React-based CRM cards that display on Contact and Company records
- Webhook subscriptions for CRM object events
- Custom workflow actions for HubSpot automation

## Authentication

Uses static authentication with OAuth scopes defined in `src/app/app-hsmeta.json`. Required scopes include contact and company read/write permissions.

## Target Platform

HubSpot Platform Version: 2025.2
